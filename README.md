# 🚀 FaithSlide：證道投影片自動化生成器 (Python)

[](https://github.com/Rye-GT500/FaitSlide)

## 🎯 專案簡述與解決的痛點

FaithSlide 是一款桌面應用程式，旨在幫助教會媒體同工**一鍵將週報（Word 格式）轉換為格式化的 PowerPoint 簡報**。

本工具將耗時的手動複製、排版、清除格式的工作，縮短至數分鐘，極大地提升了製作效率。

-----

## 🛠️ 核心功能與技術亮點

本程式結合了多種進階技術，具備高容錯性與高效率：

1.  **複雜文本解析 (Regex 驅動)：**
      * 運用 **正規表達式 (Regex)** 邏輯，精準地從 Word 粗體文本中識別並切分「主標題（一、）」、「中標題（1.）」、「小標題（1)）」以及經文引用。
      * 這確保了即使 Word 格式存在細微差異，程式也能保持準確性。
2.  **跨執行緒 UI 安全更新：**
      利用 `threading` 和 `tkinter` 的 `root.after(0, ...)` 機制，實現在背景執行網路爬蟲和文件 I/O 時，**UI 介面不凍結**，並流暢地顯示進度條和狀態信息。
3.  **系統整合與自動化：**
      * 整合 `python-docx`（讀取） $\to$ `Selenium`（網路爬取） $\to$ `python-pptx`（簡報生成）的穩定自動化管道。

## 📥 下載與使用 (For Users)

1.  請前往 **[Releases 頁面](https://github.com/Rye-GT500/FaitSlide/releases)** 下載最新版的 `FaithSlide_Automation_X.X.X.exe` 檔案。
2.  運行程式，在「PPT 自動製作」頁面選擇輸入的 Word 檔案和輸出的 PPT 路徑。

-----

## 📜 輸入文件格式規範 (重要！)

為了確保程式能正確解析內容，Word 文件中的證道部分必須使用**粗體 (Bold)**，並遵循以下標題和引用的標準格式：

### 範例輸入格式：

```
證道經文：創 1:1-3

一、這是第一個大標題： 創 1:4

1. 這是第一個中標題： 創 1:5-6 箴 5:3
1) 這是第一個小標題： 創 1:7-8, 10

2. 這是第二個中標題 創 1:11, 13, 15-17
```

### 關鍵格式說明：

| 元素 | 格式要求 | 範例 |
| :--- | :--- | :--- |
| **大標題** | 必須以 **中文數字** + **全形頓號（、）** 開頭。 | `一、這是大 標題` |
| **中標題** | 必須以 **阿拉伯數字** + **句號（.）** 開頭。 | `1. 這是中標題` |
| **小標題** | 必須以 **阿拉伯數字右括號** 開頭。 | `1) 這是小標題` |
| **所有內容**| **必須是粗體**，並且引用經文在標題後方。 | 確保 Word 檔案中所有要製作成ppt的證道內容都已設為粗體。 |
| **所有經文**| **必須是通用縮寫**| 雅 3:15|
